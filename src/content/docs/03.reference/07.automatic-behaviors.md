Baseline wires several behaviors without extra setup:

- Assets: bundles `src/assets/css/index.css` and `src/assets/js/index.js` (and other `index.js` under `src/assets/js/**`) to `dist/assets/` via PostCSS + esbuild defaults.
- Head: injects head meta through `<baseline-head>` in templates, via head-core.
- Sitemap: generates a sitemap template by default; uses `site.url` for canonical links.
- Navigator: routes registered; template off by default unless enabled via `enableNavigatorTemplate`.
- Html base: HtmlBasePlugin uses `process.env.URL` or `pathPrefix` for base href.
- Passthrough: copies `./src/static` to the output root.
- Draft preprocessor: skips `draft: true` content during `ELEVENTY_RUN_MODE=build`.
- Dev server: on-demand image generation via `eleventyImageOnRequestDuringServePlugin`.
