Baseline injects `_baseline` into global data when you register the plugin.

## `_baseline` fields

- `version`, `name`
- `verbose`
- `hasImageTransformPlugin`
- `enableNavigatorTemplate`, `enableSitemapTemplate`
- `multilingual`, `languages`
- `assets.esbuild`: `minify`, `target`

## Usage

```njk
{% raw %}
{% set opts = _baseline %}
{% if opts.enableSitemapTemplate %}
  <p>Sitemap template is on.</p>
{% endif %}
{% endraw %}
```

These values mirror the options you pass to `baseline()` and the defaults it sets. Use them for conditional UI or debugging; avoid mutating them.
