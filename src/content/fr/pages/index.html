Eleventy Plugin Baseline est un couteau suisse expérimental pour Eleventy : de petits outils d’aide préconfigurés pour les assets, le head/méta, les images et le débogage, afin de passer moins de temps à assembler toujours les mêmes éléments.

{% alertBlock "warning" %}

Work in progress: la documentation et les API peuvent évoluer.

{% endalertBlock %}

À propos

Ce plugin est ma tentative de regrouper en un kit optionnel toutes les petites fonctionnalités que j’aurais aimé voir intégrées nativement à Eleventy. Il s’appuie fortement sur les patterns et bonnes pratiques partagés par la communauté Eleventy, ainsi que par d’autres projets comme Hugo, Astro ou même WordPress.

Le plugin est actuellement disponible en preview sur npm et une version stable est prévue pour Eleventy 4.0.
