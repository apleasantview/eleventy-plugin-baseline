Eleventy Plugin Baseline prend les décisions structurelles qu'Eleventy laisse ouvertes : arborescence, pipeline d'assets, images, SEO, sitemaps. Vous passez la mise en place et commencez à construire.

{% alertBlock "warning" %}

Work in progress : la documentation et les API peuvent évoluer.

{% endalertBlock %}

À propos

Si vous avez déjà démarré un nouveau projet Eleventy et vous êtes retrouvé à câbler les mêmes éléments pour la troisième fois — arborescence, moteur de templates, formats d'image, balises meta, bundling d'assets, sitemap — c'est pour vous. Baseline prend ces décisions ensemble, pour qu'elles s'articulent entre elles.

Votre site reste le vôtre. Baseline gère l'infrastructure. Votre design, votre contenu, ce qui fait de votre site le vôtre — tout ça reste à vous.

Le plugin est disponible en preview sur npm. Une version stable est prévue pour Eleventy 4.0.
